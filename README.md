▶️ Youtube Ad-Blocked app

For compiling on your end
run command

```sh
$ npm i
```

For Running this you must need NPM and Node Installed.

<a href="https://nodejs.org/en/">Node.js</a>
and then run 

```sh
$ electron-forge make
```
